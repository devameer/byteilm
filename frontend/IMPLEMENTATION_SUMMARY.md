## üí° **Tips for Developers**

1. **Consistency**: Always use design system classes from `app.css`
2. **Performance**: Prefer CSS transitions over JavaScript animations
3. **Accessibility**: Test with keyboard navigation and screen readers
4. **Responsive**: Test all breakpoints during development
5. **Documentation**: Update `DESIGN_SYSTEM.md` when adding components
6. **Colors**: Use semantic tokens (primary, accent) not raw colors
7. **Spacing**: Follow the spacing scale (4, 6, 8, 12, 16, 24, etc.)
8. **Typography**: Use weight scale consistently (400, 600, 700, 900)
